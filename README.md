# base-rate

Репозиторий с двумя независимыми мини-приложениями (каждое — один HTML-файл, работают офлайн, без сборки).

## ✅ Мои задачи — `tasks.html`

Простое и удобное приложение для ведения личных задач с отслеживанием выполнения и напоминаниями.

**Возможности:**
- Добавление задач с заметкой, сроком (дата + время) и приоритетом
- Отметка выполнения в один клик, прогресс-бар и статистика
- Фильтры: Все / Активные / Сегодня / Просроченные / Готово
- Напоминания через системные уведомления браузера по наступлению срока
- Подсветка просроченных и скоро истекающих задач
- Данные хранятся локально (`localStorage`) — ничего не уходит в сеть
- Тёмная тема (по настройкам системы), адаптивно для телефона

**Как пользоваться:** откройте `tasks.html` в браузере. Разрешите уведомления, чтобы получать напоминания.
